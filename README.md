# Glassdoor Unlocked

**Glassdoor Unlocked** is a Tampermonkey userscript that enhances your Glassdoor experience by removing intrusive overlays and revealing collapsed reviews and interview details.

## Features

- **Overlay Removal**: Hides promotional overlays that obstruct content.
- **Expanded Reviews**: Reveals full review text by removing collapse states.
- **Interview Insights**: Unfolds detailed interview feedback for comprehensive insights.
- **Dynamic Updates**: Automatically adjusts to changes on the page for smooth functionality.

## Installation

### Using Tampermonkey

1. **Install Tampermonkey**:
   - [Chrome](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmipfmgdfohmicgkkbigl/) 
   - [Firefox](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)
   - [Microsoft Edge](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/igfpebdbmgbkggkoeafcafbcaeneebkk)

2. **Add the Script**:
   - Click on the Tampermonkey icon in your browser toolbar.
   - Select "Dashboard."
   - Click the "+" icon to create a new script.
   - Remove any template code in the new script window.
   - Copy and paste the script code from [glassdoor_unlocked.js](https://github.com/h3x0x0/Glassdoor-Unlocked/blob/main/glassdoor_unlocked.js) into the Tampermonkey editor.

3. **Save the Script**:
   - Click "File" then "Save" or press `Ctrl + S`.

4. **Visit Glassdoor**:
   - Navigate to any Glassdoor review or interview page to see the script in action.

### Using Other Userscript Managers

1. **Install a Userscript Manager** that supports your browser, such as:
   - [Violentmonkey](https://violentmonkey.github.io/)
   - [Greasemonkey](https://addons.mozilla.org/en-US/firefox/addon/greasemonkey/)

2. **Add the Script**:
   - Follow similar steps as above to create a new script and paste in the code.
